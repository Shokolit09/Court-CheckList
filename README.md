# צ'קליסט בית משפט - מערכת תיעוד דיונים

## תיאור
מערכת צ'קליסט לקלדניות בית משפט, המלווה את הקלדנית לאורך כל תהליך העבודה בדיון.

## התקנה

### דרישות מקדימות
- Node.js (גרסה 18 ומעלה)
- npm

### שלבי התקנה

1. **פתח את התיקייה ב-WebStorm**

2. **התקן את החבילות:**
   ```bash
   npm install
   ```

3. **הרץ את האפליקציה במצב פיתוח:**
   ```bash
   npm start
   ```

4. **בנה קובץ EXE להפצה:**
   ```bash
   npm run dist
   ```
   הקובץ יווצר בתיקיית `dist`

## שימוש

- האפליקציה תעלה אוטומטית בצד ימין של המסך
- כל שלב דורש אישור לפני מעבר לשלב הבא
- לחיצה על "בוצע" מסמנת השלמת שלב
- לחיצה על "מעקף" מאפשרת דילוג על שלב עם ציון סיבה
- "התחל דיון חדש" מאפס את כל הצ'קליסט

## הגדרת הפעלה אוטומטית עם Windows

האפליקציה מוגדרת להיפתח אוטומטית עם הפעלת המחשב.
אם זה לא עובד, ניתן להוסיף קיצור דרך ידנית לתיקייה:
```
C:\Users\[שם משתמש]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
```
